# **App Name**: HWHS Exam Portal

## Core Features:

- Welcome Page: Display school logo and name on the home page.
- Student Registration: Allow students to register with name, roll number, class, and section.
- Student Login: Enable students to log in and access their profile.
- Faculty Login: Enable faculty to log in and manage exams.
- Exam Scheduling: Allow faculty to schedule exams for different classes and subjects.
- MCQ Exam: Display MCQ questions with a timer and options, submission tracked and sent via email.
- Code Evaluation Tool: Feature the ability for faculty to grade the freeform coding section.  Use an LLM tool to provide assistance to teachers as they manually grade student work. Each step is submitted for AI review, and the review presented in order to incorporate (or ignore).

## Style Guidelines:

- Primary color: Dark blue (#224B7A), a professional color evoking stability and intelligence, aligning with an academic environment.
- Background color: Light gray (#E0E5EA), a subtle hue complementing the blue without causing distraction.
- Accent color: Light orange (#EBB072) to add contrast.
- Headline font: 'Space Grotesk', sans-serif with a technical feel for headlines; Body font: 'Inter' as sans-serif and grotesque style with a modern neutral look; consider code display via 'Source Code Pro' monospace font if useful
- Use academic-themed icons for navigation and categories.
- Clean and structured layout with a focus on readability and ease of navigation.
- Subtle transitions and loading animations to enhance user experience.
